# Uagents Scripts

Lagents related scripts for Blackbox4.

## Scripts

- `autonomous-loop.sh`
- `lib.sh`

## Usage

From .blackbox4 root:
```bash
./4-scripts/agents/<script>.sh [args]
```

From within 4-scripts:
```bash
cd 4-scripts
./agents/<script>.sh [args]
```
