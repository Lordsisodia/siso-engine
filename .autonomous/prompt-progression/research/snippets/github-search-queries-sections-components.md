# GitHub Search Queries — Page Sections / Components (FAQ, pricing, testimonials, newsletter)

Use this query bank when you want to discover **copy/paste friendly page sections** we can mine into our own
component library and Blocks Kit:
- FAQ accordions, pricing tables, testimonials
- newsletter / waitlist / CTA blocks
- feature grids, stats, comparison tables
- blog index primitives (post cards, tag chips) *only when it shows up inside section libraries*

Recommended run mode:
- use `--no-derived-queries` (avoid competitor-feature-map query pollution)
- keep `--min-stars` >= 100 for signal (lower only if you need niche stacks)
- use `--exclude-regex` to avoid personal sites/portfolios/tutorial repos
