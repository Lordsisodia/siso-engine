# GitHub Search Queries — Excel/Sheets paste + clipboard TSV edge cases (v1)
#
# Goal: utilities and examples for handling “paste from Excel/Google Sheets” into grids:
# - tab-separated clipboard parsing (TSV)
# - quoted fields with newlines/tabs
# - normalization of ragged rows / trailing empty columns
# - mapping headers to schema fields
# - xlsx helpers (when users upload files instead of pasting)
#
# This complements PapaParse / csv-import / zod-csv from previous passes.

## Clipboard TSV parsing (Excel/Sheets paste)
- `excel paste tsv parse quotes newlines`
- `google sheets paste tsv parse`
- `parse clipboard tsv with quotes`
- `tab separated clipboard parser typescript`
- `clipboard spreadsheet paste parser`

## SheetJS / xlsx helpers (file import)
- `sheetjs xlsx parse typescript`
- `xlsx parse to json typescript`
- `sheetjs parse xlsx to rows`

## ExcelJS helpers (Node/browser)
- `exceljs parse xlsx typescript`
- `exceljs import xlsx validation`

## Header mapping + validation patterns
- `map csv headers to schema zod`
- `import spreadsheet header mapping typescript`
- `validate spreadsheet rows zod`
