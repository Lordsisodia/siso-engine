# GitHub Search Queries — Store credit / Gift cards / Loyalty

Goal: find OSS primitives for:
- gift card issuance + redemption (balance ledger, expiry, partial redemption)
- store credit (refund-to-credit, credit memo, wallet balance)
- loyalty points (earn/redeem, tiers) **without** drifting into crypto tokens

Avoid:
- crypto/web3 “wallet” repos
- payment processing (“credit card”) repos
