# GitHub Search Queries — Blog + Content Blocks (Targeted)

Use this query bank to find **content primitives** we can mine for blog/editorial pages and
“content + commerce” pages:
- MDX/Markdown rendering components
- TOC + anchors + reading time
- code blocks (Shiki/rehype-pretty-code style)
- reusable marketing blocks (FAQ/pricing/testimonials/newsletter)

Recommended run mode:
- `--no-derived-queries`
- use an `--exclude-regex` to avoid personal sites/portfolios
